命令通用

mongoexport

mongoimport

mongodump 导出二进制bson结构的数据及其索引信息

  1. 导出的文件放在以database命名的目录下

  2. 每个表导出2个文件,分别是bson结构的数据文件, json的索引信息

  3. 如果不声明表名, 导出所有的表

mongorestore -h IP –port 端口 -u 用户名 -p 密码 -d 数据库 –drop 文件存在路径 –drop的意思是，先删除所有的记录，然后恢复
